# WeatherTestApp

Создать приложение с использованием https://openweathermap.org или подобного АПИ на два экрана:
# Требования
# На первом экране:
- таблица со списком городов, можно захардкодить штук 5-6
- как доп. задача можно запрашивать геолокацию и показывать дополнительно текущее место положение первым в списке
- по нажатию на город открыть второй экран

# Второй экран:
- таблица в которой показана погода в выбраном городе на промежуток времени, например на 10 дней с температурой, формат произвольный
- если отсутствует соединение с интернетом показать последние загруженные данные или экран с сообщением например “сохранённые данные отсутствуют”
- погода должна хранится в базе данных и обновляться соотвественно из нее же
- PullToRefresh для обновление таблицы

Для БД использовать Realm, в остальном нет ограничений по библиотекам
